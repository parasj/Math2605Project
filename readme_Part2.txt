1. Encoding
	To run the encoding program, run in the Matlab shell: driver_encode
	This will launch a GUI where you enter n, the length of the input stream, and it will generate a random input stream x and compute y, the output stream. x and y will be displayed in a popup and a0 and a1 will be printed to the shell.

2. Decoding
	To run the decoding program, run in the Matlab shell: driver_decode
